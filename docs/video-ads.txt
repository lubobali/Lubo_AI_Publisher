==============================================================================
LUBOT VIDEO ADS — MASTER PLAN
"Slow. Luxury. Best quality possible." (Lubo, Jul 2026)
==============================================================================

GOAL
  Produce 3-5 short, cinematic demo videos of LuBot in action across its 3 real
  modes (stock, website, my-files), narrated in Lubo's CLONED voice over premium
  music, wrapped in the LuBot blue-steel brand. These are top-of-funnel lead ads:
  every video ends by sending the viewer to lubot.ai. Ties to the north star
  (market LuBot, gain paid users) and the "market lead engine" direction.

ETHOS
  Not fast, not cheap, not templated-looking. Each video should feel like a
  product film from a company 100x our size. One-time batch project: build the
  system once, shoot all videos in one push, then it is done.

QUALITY BAR (the "luxury" checklist every video must pass)
  [ ] Footage is crisp (4K source), smooth zooms, no jitter, no dead air.
  [ ] Voice is indistinguishable from Lubo (Professional clone), clean, paced.
  [ ] Music is cinematic + on-brand, ducked under the voice, never fighting it.
  [ ] Brand bookends (intro + outro) match the LinkedIn cards exactly.
  [ ] Captions burned in, styled, readable MUTED (most people watch with no sound).
  [ ] Color graded (consistent look across all videos).
  [ ] One clear CTA: lubot.ai. Every time.
  [ ] The 3 second hook earns the next 3 seconds. No slow starts.

THE STACK (locked picks — best quality, 2026)
==============================================================================
  Script          : Claude (drafts + tightening, knows LuBot + Lubo voice)
  Voice           : ElevenLabs — PROFESSIONAL Voice Clone (not instant)
  Screen footage  : Screen Studio (Mac) — auto-zoom, cursor smoothing, 4K
  Music           : Epidemic Sound (monthly) OR Artlist — one signature family
  Brand frames    : LuBot card engine (cards.py) — intro/outro/lower-thirds
  Edit + color    : DaVinci Resolve (free, pro-grade)
  Captions        : DaVinci / CapCut auto + brand styling
  Optional b-roll : Runway / Kling — cinematic transition shots, sparingly
  Assets/backup   : Local folders + a cloud drive; download EVERYTHING in-month

COST STRATEGY — ONE MONTH IS ENOUGH
  Subscribe -> batch all videos -> DOWNLOAD every voice file + music track ->
  cancel. Licenses on downloaded/generated assets persist.
  ~$22 ElevenLabs (1 mo) + ~$15 Epidemic (1 mo) + ~$89 Screen Studio (one-time)
  + DaVinci free = under ~$130 total, then cancel the two subs.
  RULE: nothing gets produced after the subs lapse, so finish the batch first.

THE VIDEOS (the batch)
==============================================================================
  V1  INTRO / SIZZLE (15-20s)   — "Meet LuBot. One agent, three modes." brand-forward.
  V2  STOCK mode (25-40s)       — ask a real market question -> LuBot answers with data.
  V3  WEBSITE mode (25-40s)     — point it at a site -> it does the thing live.
  V4  MY-FILES mode (25-40s)    — drop in files -> ask -> grounded answer from YOUR docs.
  V5  (optional) FOUNDER note   — Lubo on camera / voice: why he built it. Trust.
  Per-mode color identity: stock=blue, website=teal, files=cyan (from card worlds).

PER-VIDEO PIPELINE (repeatable assembly line)
  1. SCRIPT (20-40s): Hook -> the pain -> LuBot does it live -> the result -> CTA.
  2. RECORD the real demo in Screen Studio (clean run, 4K, one mode = one story).
  3. VOICE the script in ElevenLabs (Lubo clone), export WAV.
  4. ASSEMBLE in DaVinci: footage + voice + music (duck under voice) + intro/outro
     + lower-thirds + captions.
  5. GRADE + mix audio + export: 9:16 (feed) and 16:9 (YouTube/site).

==============================================================================
  0.1  Lock the goal + audience + the ONE CTA (lubot.ai). Written down.
  0.2  Buy/setup tools: Screen Studio (one-time), ElevenLabs Creator (1 mo),
       Epidemic Sound (1 mo), DaVinci Resolve (free). Create accounts.
  0.3  VOICE CLONE — ElevenLabs Professional:
       - Record ~30 min of clean audio (quiet room, good mic, consistent tone).
       - Read varied, natural sentences (Claude provides the script/checklist).
       - Train the Professional clone. Test it on one paragraph, tweak stability/
         style until it is unmistakably Lubo. (Do ALL narration this month.)
  0.4  MUSIC — pick ONE signature track family on Epidemic (2-3 tracks, same mood:
       modern, cinematic, premium, not hype-y). Download them. This gives the set
       a consistent sonic brand.
  0.5  BRAND SYSTEM — from cards.py, generate the reusable video assets:
       - Intro card/animation (blue-steel, logo, "LuBot", lubot.ai).
       - Outro card/animation ("Try it -> lubot.ai").
       - Lower-third template (name/mode label, per-mode color).
       - Caption style (font Grotesk, brand colors, position). LOCK these.
  0.6  Decide formats + specs: 9:16 (1080x1920) primary for feed, 16:9 secondary.
       Frame rate 30fps. Loudness target for voice. Safe margins for captions.
  0.7  Set up the FOLDER STRUCTURE (one folder per video: /script /screen /voice
       /music /exports) + a cloud backup. Download-everything discipline.

  4.5  Color: one grade across all. Export final masters + feed cuts.
